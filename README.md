# bootstrapping

Install ruby
Install rails
Install firefox (for tests)
Install geckdriver (for tests)

```
bundle install
bundle exec rake db:setup
bundle exec rake db:migrate
bundle exec rake db:seed
```

# running

```
rails s
http localhost:3000
```
